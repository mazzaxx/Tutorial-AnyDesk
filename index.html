Tutorial para Retirar Janela de Licença do AnyDesk

1. Desinstalar o AnyDesk

Para remover a instalação atual do AnyDesk e suas configurações, siga estes passos:

  1. Acesse as configurações do computador:
    1. Pressione Windows + I para abrir as Configurações.
    2. Vá até Aplicativos e encontre AnyDesk na lista.
    3. Clique em Desinstalar e siga as instruções na tela.
  2. Se aparecer uma janela do Windows pedindo usuário e senha:
    1. Digite .\usuario no campo de usuário e pressione Enter.
    2. Deixe o campo de senha em branco e pressione Enter.
  3. Após isso, aparecerá uma janela do AnyDesk com a opção Apagar Configurações.
    1. Marque a caixa Apagar Configurações e confirme.

2. Passo a Passo para Limpeza

É importante limpar os arquivos temporários e as pastas relacionadas ao AnyDesk para garantir uma nova instalação limpa:

  1. Abrir a Pasta Temp:
    1. Pressione as teclas Windows + R para abrir a janela "Executar".
    2. Digite temp e pressione Enter.
  2. Excluir os Arquivos: A pasta C:\Users\SeuUsuário\AppData\Local\Temp será aberta.
    • Selecione todos os arquivos (pressione Ctrl + A).
    • Pressione Delete no teclado e confirme a exclusão, se necessário.
  3. Limpar a Pasta %temp%:
    1. Pressione Windows + R novamente para abrir a janela "Executar".
    2. Digite %temp% e pressione Enter.
    3. Repita o processo de exclusão como na etapa anterior.
  4. Acessar a Pasta %appdata%:
    1. Pressione Windows + R novamente para abrir a janela "Executar".
    2. Digite %appdata% e pressione Enter.
    3. A pasta C:\Users\SeuUsuário\AppData\Roaming\AnyDesk será aberta.
    4. Selecione todos os arquivos dentro da pasta e pressione Delete.

3. Download do AnyDesk

Clique no botão abaixo para baixar a versão recomendada do AnyDesk:

ANYDESK 6.3

Ao clicar, você será redirecionado para uma página do Dropbox. Clique em "Baixar" no meio da tela. Se aparecer uma tela de login, ignore e clique em "Continuar apenas com o download".

Se preferir, pode instalar a versão mais recente do AnyDesk, mas a versão antiga apresenta menos problemas e geralmente demora mais para voltar a aparecer a janela de licença.

4. Reinstalar o AnyDesk

Após baixar a versão do AnyDesk, siga estes passos para reinstalá-lo:

  1. Abra a pasta de Downloads no seu computador.
  2. Localize o executável do AnyDesk que você baixou.
  3. Ao abrir o executável, aparecerá a mensagem em vermelho: Instalar AnyDesk neste dispositivo.
  4. Clique nessa opção.
  5. Marque a opção Desabilitar atualizações automáticas.
